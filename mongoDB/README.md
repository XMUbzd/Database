## mongoDB云
- 登录上述网址，进入注册界面。填写邮箱，用户名，密码等。
- 完成各个选项后（基本不用改，选Free就完事了）
- 创建Cluster，按照步骤和个人的需求
- 连接Cluster，能得到一个Client开头的代码Copy就可以使用了
- tip：注意挂代理的要把自己的端口暴露出来，ip地址设为0.0.0.0。不然会造成连接的失败。
- 懂的都懂（嚣张脸）
